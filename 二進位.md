# 進位轉換
# 例子:10010010
# 一.1,s 補數
### Step.1 判斷正負數
```
0 -> 正
1 -> 負

[1]0010010 -> 負數

```

### Step.2 反向

```
由0變1
由1變0

10010010 -> 01101101
```
### Step.3 計算轉換

```
由右往左看
二進位轉十進位
01101101 -> 1+4+8+32+64 = 109

```

### Step.4 計算結果加上正負
```
[-] + [109] = [-109]
```
